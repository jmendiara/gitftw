# Contributing to gitftw

1. Make a fork

2. Test the code!

```sh
npm run test
```

3. Check your coverage!

```sh
npm run coverage
```

4. Lint your code!

```sh
npm run lint
```

5. Check the code documentation is fine!

```sh
npm run doc
```

6. Try to squash your commits (optional)

7. Make a PR to the master branch

8. Thanks a lot!
